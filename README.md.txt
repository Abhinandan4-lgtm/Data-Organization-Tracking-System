# Data Organization & Tracking System

## Overview

A data management project developed using Python, Pandas, and Microsoft Excel to organize, clean, and track records efficiently.

## Features

- Import CSV datasets
- Remove duplicate records
- Handle missing values
- Standardize data entries
- Generate cleaned datasets
- Create department-wise reports
- Build Excel dashboards using Pivot Tables

## Technologies Used

- Python
- Pandas
- Microsoft Excel
- Google Sheets
